# P8-008: Social Profile Tab

## Description
Create a new "Profile" tab in the main navigation or a dedicated profile section. This view will serve as the hub for the user's social identity in the app.

## Requirements
- [x] Create a `ProfilePage` component.
- [x] Display user's name, email, and avatar (placeholder or generated).
- [x] Show a summary of social stats (e.g., "5 Friends", "3 Shared Goals").
- [x] Add a tabbed interface within the profile: "Overview", "Friends", "Settings".

## Technical Details
- Route: `/profile`
- Components: `ProfileHeader`, `ProfileStats`.
